# xPack Windows Build Tools Archive

This repository contains a packaged Windows build-tools archive used by embedded development workflows.

## Repository contents

| File | Description |
| --- | --- |
| `xpack-windows-build-tools-4.3.0-1-win32-x64.zip` | xPack Windows Build Tools 4.3.0-1 archive for 64-bit Windows |

## Usage

1. Download the ZIP archive.
2. Extract it to a path without special characters or unnecessary spaces.
3. Add the required tool directories to your development environment or configure your IDE/build system to use their absolute paths.
4. Open a new terminal and verify the required executable versions before building a project.

## Notes

- This repository contains a packaged tool archive, not its source code.
- The archive may be retained for compatibility with historical Geehy development projects.
- Confirm that your project requires this exact version before using it.
- For newer projects, use the tool versions specified by the corresponding SDK, example project, or setup guide.
- Review applicable third-party licenses included in the archive before redistribution.
